STEP1
너는 교과서 분석 전문가다.

아래 PDF 내용을 OCR 기반으로 분석해서 단순 텍스트가 아니라
"구조화된 학습 데이터"로 변환해라.

요구사항:
1. 모든 텍스트를 빠짐없이 추출
2. 표는 반드시 구조를 유지해서 정리
3. 그림/도표는 내용 추론해서 설명 추가
4. 핵심 개념은 별도로 표시

출력 형식(JSON):
{
  "chapters": [
    {
      "title": "",
      "concepts": [],
      "details": "",
      "tables": [],
      "figures": [
        {
          "description": "",
          "related_concept": ""
        }
      ]
    }
  ]
}


STEP2

너는 출제위원이다.

위 데이터를 기반으로 실제 시험에서 나올 가능성이 높은
핵심 개념과 출제 포인트를 정리해라.

조건:
- 암기형 / 이해형 / 계산형 구분
- 자주 틀리는 포인트 강조
- 표/그림 기반 문제 가능 포인트 따로 표시

출력:
1. 핵심 개념 요약
2. 출제 포인트 리스트
3. 함정 포인트



STEP3

너는 실제 시험 출제자다.

위 분석 내용을 기반으로 기출 문제 스타일로 문제를 만들어라.

조건:
- 객관식 5지선다
- 난이도: 쉬움 / 보통 / 어려움 섞기
- 표/그래프 해석 문제 반드시 포함
- 단순 암기 문제 금지

출력:
문제
보기
정답
해설 (왜 틀리는지 포함)

각단계별 MD 파일로 결과물 생성
