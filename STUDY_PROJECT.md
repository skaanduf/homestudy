## 🛠️ 기술 스택
- **프론트엔드**: Vite + React + TypeScript
- **스타일링**: Vanilla CSS (Premium 디자인 적용)
- **배포**: GitHub Pages (GitHub Actions 자동화)
- **AI**: Google Gemini (교과서 요약 및 분석)

## 🔄 표준 작업 프로세스 (Standard Pipeline)

새로운 교과서나 단원을 추가할 때는 다음 4단계를 따릅니다.

1.  **[다운로드]**: `streamdocs_download.py`를 실행하여 교과서 PDF를 확보합니다.
2.  **[텍스트 추출]**: `pdf_to_text.py`를 실행하여 PDF에서 텍스트(.txt)를 추출합니다. (OCR)
3.  **[요약]**: 추출된 텍스트를 Gemini에게 전달하여 핵심 내용을 요약합니다.
4.  **[웹 제작]**: 요약된 내용을 `study-web` 프로젝트의 컴포넌트로 변환하여 반영합니다.
