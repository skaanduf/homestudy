# StreamDocs PDF 다운로더 사용법

## 준비 (최초 1회)

```bash
pip3 install --break-system-packages Pillow img2pdf
```

---

## 사용법

### 기본 (페이지 수 자동 감지)

```bash
python3 streamdocs_download.py <streamdocsId>
```

### 전체 옵션

```bash
python3 streamdocs_download.py <streamdocsId> <총페이지수> <출력파일명.pdf>
```

---

## streamdocsId 찾는 방법

StreamDocs 뷰어 URL에서 `streamdocsId=` 뒤의 값을 복사한다.

예시 URL:
```
https://view.chunjae.co.kr/streamdocs/view/sd;streamdocsId=bF5lHkVKdoKN8-Nyppwar3tWOreEqjLjULDJMTyWt94;isExternal=eQ;...
```

→ streamdocsId = `bF5lHkVKdoKN8-Nyppwar3tWOreEqjLjULDJMTyWt94`

---

## 예시

```bash
# 페이지 수 자동 감지, 출력 파일명 자동 생성
python3 streamdocs_download.py bF5lHkVKdoKN8-Nyppwar3tWOreEqjLjULDJMTyWt94

# 페이지 수와 파일명 직접 지정
python3 streamdocs_download.py bF5lHkVKdoKN8-Nyppwar3tWOreEqjLjULDJMTyWt94 125 천재_사회1.pdf
```

---

## 참고

- 다운로드 중단 후 재실행하면 이미 받은 페이지는 자동으로 건너뜀 (캐시)
- 임시 파일은 `/tmp/streamdocs_<id앞12자리>/` 에 저장됨
- zoom=150 기준 고화질 PNG로 저장 후 PDF로 합침
